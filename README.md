# 💻 Etec - Projetos e Exercícios Web

Este repositório contém exercícios práticos desenvolvidos durante o primeiro módulo do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. Os projetos focam no aprendizado de **HTML5** e **CSS3**, explorando desde a criação de formulários interativos até a organização de dados estruturados.

---

## 🚀 Projetos em Destaque

### 1. Formulário de Cadastro (Layout com Table)
Neste exercício, utilizei tabelas para garantir um alinhamento preciso de uma interface de cadastro completa.
* **Estrutura:** Organização de campos de Nome, Email, Telefone, Sexo (radio buttons) e Mensagem (textarea) dentro de `<table>`.
* **Funcionalidades:** Uso do atributo `required` para validação básica e `colspan="2"` para centralizar os botões de ação.
* **Estilização:** Botões personalizados com IDs `#enviar` (verde) e `#limpar` (vermelho), além de efeitos de `hover` prata nos campos de texto.

### 2. Formulário Temático (Digimon)
Um projeto focado em design de interface (UI) e interatividade visual.
* **Design Visual:** Aplicação de imagem de fundo personalizada utilizando `background-size: cover` e centralização de conteúdo com `text-align: center`.
* **Interatividade:** Botões que alternam cores no estado `:hover`, passando de fundo preto para branco com texto verde.
* **Campos:** Implementação de campos específicos para `email`, `text` (nickname) e `password`.

### 3. Folha de Pagamento (Tabela de Dados)
Foco em HTML semântico para a exibição de dados financeiros estruturados.
* **Semântica:** Uso das tags `<thead>`, `<tbody>`, `<th>` e `<caption>` para uma estrutura acessível.
* **Manipulação de Células:** Uso de `colspan="3"` para a criação da linha de "Total" na base da tabela.
* **CSS Avançado:** Aplicação de `border-collapse: collapse` e efeitos de destaque em células individuais através do seletor `td:hover`.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação de documentos e formulários.
* **CSS3:** Estilização, pseudo-classes (`:hover`) e manipulação de backgrounds.
