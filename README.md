# 💻 Etec - Projetos e Exercícios Web

Este repositório contém os projetos e exercícios práticos desenvolvidos durante o 1º semestre do curso de **Desenvolvimento de Sistemas (DS)** na **Etec Joaquim Ferreira do Amaral - Jaú**.

O objetivo deste espaço é documentar minha evolução no aprendizado de tecnologias front-end e estruturação de dados para a web.

---

## 🚀 Projetos em Destaque

### 1. Formulário de Cadastro (Tema: Jogos/Digimon)
Interface desenvolvida para simular o cadastro de usuários em uma plataforma de jogos.
* **Foco:** Prática de elementos de formulário (`<input>`) e estilização visual com CSS3.
* **Destaque:** Aplicação de conceitos de UI básica para criar uma interface temática.

### 2. Tabela de Folha de Pagamento
Simulação de uma estrutura de dados organizacional focada em cálculos financeiros.
* **Foco:** Estruturação de tabelas complexas (`<table>`, `<thead>`, `<tbody>`) e alinhamento de dados com CSS.
* **Diferencial:** Demonstra organização lógica e atenção aos detalhes em projetos que lidam com dados estruturados.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica de conteúdo e formulários.
* **CSS3:** Estilização básica, design visual e layout.
* **Git & GitHub:** Versionamento de código e documentação de progresso.
